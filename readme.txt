--- To setup on a new machine ---

Install Python 3.11.1
Install Pycharm Community Edition
Open the command terminal and navigate to your "GitHub/english-game" directory
Create a new virtual environment in the english-game directory entitled .venv (type the command "python -m venv .venv")
Activate it by navigating to english-game/.venv/Scripts and running activate.bat from the terminal
Run "pip install -r requirements.txt" to install all necessary modules
Open Pycharm, press Ctrl+Alt+S and add .venv/Scripts/python.exe to your interpreters for the english-game project
The game should run now
