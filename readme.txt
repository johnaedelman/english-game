--- To setup on a new machine ---

Install Python 3.11.1
Create a new venv in the english-game directory entitled .venv (command is "python -m venv .venv" while in the english-game directory.)
Activate it by running activate.bat from the terminal
Run "pip install -r requirements.txt" to install all modules
Ctrl+Alt+S and add the .venv/Scripts/python.exe to your interpreters
Should run now
